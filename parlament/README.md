# Платформа рейтинга активности для молодежного парламента

## Описание проекта
Веб-платформа для автоматизации учета активности участников молодежного парламента и формирования прозрачного рейтинга для отбора кандидатов на стажировки и должности.

## Функциональность
- ✅ Регистрация и авторизация пользователей
- ✅ 4 роли: участник, организатор, администратор, кадровая служба
- ✅ Создание и модерация мероприятий
- ✅ Начисление баллов с учетом коэффициента сложности
- ✅ Глобальная таблица лидеров с фильтрацией по направлениям
- ✅ Профили пользователей с портфолио достижений
- ✅ Дашборд активности с графиками
- ✅ Панель администратора для управления пользователями
- ✅ Инспектор кадрового резерва с фильтрами и экспортом в PDF

## Технологии
- PHP 7.4+
- MySQL 5.7+
- HTML5/CSS3
- JavaScript (Chart.js для графиков)

## Установка и запуск

### Требования
- Веб-сервер (Apache/Nginx)

- MySQL 5.7 или выше
